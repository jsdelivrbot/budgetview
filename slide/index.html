คำแนะนำในการกรอกโครงการ

หน้าต่างหลัก

A 1. เพิ่มโครงการใหม่เข้าระบบ

เพิ่มได้ 2 ประเภท คือ
1. โครงการประเภทแผนบูรณาการพัฒนาพื้นที่ระดับภาค
2. โครงการประเภทแผนบูรณาการส่งเสริมการพัฒนาจังหวัดและกลุ่มจังหวัด

A 2. ขั้นตอนในการกรอกข้อมูลโครงการ

ต้องกรอกให้ครบทั้ง 5 ขั้นตอน

A 3. วิธีลงข้อมูลพื้นที่โครงการ


แบบตามขอบเขตการปกครอง

แบบพื้นที่จำเพาะ

A 3.1 ลงข้อมูลพื้นที่โครงการ
ตามขอบเขตการปกครอง



ใช้กับกรณีอ้างอิงสถานที่หรือพื้นที่ดำเนินการตามขอบเขตการปกครอง

A 3.2 ลงข้อมูลพื้นที่โครงการ
แบบพื้นที่จำเพาะ



โดยสามารถลงพื้นที่ได้ 3 รูปแบบ ได้แก่
1. เส้น เช่น โครงการสร้างถนน
2. รูปหลายเหลี่ยม เช่น โครงการสร้างอ่างเก็บน้ำ
3. จุด แทนพิกัดพื้นที่ที่ไปดำเนินการ เช่น ตำแหน่งบ้านผู้สูงอายุ

A 4. เรียกดูข้อมูลสรุปโครงการของตนเอง



สามารถค้นหาของโครงการของตนเอง โดยการเลือกพื้นที่ หรือยุทธศาสตร์ต่าง ๆ

สำหรับแอดมิน

หน้าต่างหลัก


สามารถเรียกดูได้ทุกโครงการที่กรอกมาในระบบ

B 1. ดูข้อมูลในรูปแบบแผนที่


สามารถดูได้ 3 รูปแบบ ได้แก่
1. จำนวนโครงการในพื้นที่
2. จำนวนงบประมาณในพื้นที่
3. การครบจำนวนห่วงโซ่คุณค่า

B 1.1 แผนที่แสดงจำนวนโครงการทั้งหมด


สีของแผนที่แสดงจำนวนโครงการทั้งหมดที่ของบประมาณทำในพื้นที่

B 1.2 แผนที่แสดงจำนวนงบประมาณทั้งหมด


สีของแผนที่แสดงจำนวนงบประมาณโครงการทั้งหมดที่ของบประมาณทำในพื้นที่

B 1.3 แผนที่แสดงห่วงโซ่คุณค่า


แผนที่จะแสดงสีของห่วงโซ่ คุณค่าในพื้นที่ว่ามีครบตามห่วงโซ่หรือไม่
โดยจะแสดงออกเป็น 3 สี คือ

สีแดง มี 1 ห่วงโซ่

สีเหลือง มี 2 ห่วงโซ่

สีเขียว มีครบทั้ง 3 ห่วงโซ่

B 2. ตรวจสอบห่วงโซ่คุณค่าในพื้นที่



คลิกลงในแผนที่จะแสดงข้อมูลกราฟ
จำนวนห่วงโซ่คุณค่าของโครงการ

คลิกเลือกดูจำนวนห่วงโซ่คุณค่าโครงการในระดับภาคหรือระดับจังหวัด

B 3. ตรวจสอบรายชื่อโครงการในพื้นที่



คลิกเลือกดูรายชื่อโครงการในระดับภาคหรือระดับจังหวัด

B 4. เปรียบเทียบกับพื้นที่อื่น



สามารถตรวจสอบหรือเปรียบเทียบข้อมูลโครงการในพื้นที่ที่ต่างกันได้

เรียกดูชั้นข้อมูลสนับสนุน

C 1. ชั้นข้อมูลแผนที่



แผนที่จะมีชั้นข้อมูลต่าง ๆ ไว้สำหรับสนับสนุนการตัดสินใจ
โดยจะมีทั้งหมด 5 ประเภทด้วยกัน คือ
1. น้ำ
2. ที่ดิน
3. การใช้ที่ดิน
4. เกษตร
5. สถานที่สำคัญ

C 2. เรียกดูแผนที่โครงการที่ของบประมาณ



1. เรียกดูตามขอบเขตการปกครอง
2. เรียกดูตามพื้นที่จำเพาะ

C 3. เรียกดูกราฟสรุปโครงการ



กราฟจะแสดงจำนวนโครงการที่จำแนกในยุทธศาสตร์ต่าง ๆ

C 4. แสดงรายชื่อโครงการทั้งหมด



1. คลิกเลือกรายชื่อโครงการในพื้นที่
2. คลิกดูรายละเอียดของโครงการ

C 5. รายละเอียดโครงการและพื้นที่ดำเนินการ



จะแสดงข้อมูลรายละเอียดโครงการ รวมไปถึงแสดงแผนที่ตำแหน่งโครงการที่ดำเนินการในพื้นที่

C 6. สืบค้นโครงการ


สามารถค้นหาในรายพื้นที่หรือค้นหาข้อมูลตามยุทธศาสตร์ต่าง ๆ
